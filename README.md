## 🏁 Getting Started <a name = "getting_started"></a>

- clone project
  ```
  git clone https://github.com/harshraj8843/todo_app_DJANGO
  ```
  ```
  cd todo_app_DJANGO/
  ```
- install requirements
  ```
  pip install -r requirements.txt
  ```
- migrate
  ```
  python manage.py migrate
  ```
- runserver
  ```
  python manage.py runserver
  ```
